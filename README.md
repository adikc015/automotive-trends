# Exploring Trends in the Automotive Industry

This project contains a Tableau workbook and its source dataset for analyzing trends in the automotive industry.

## Project Files

- Exploring Trends in the Automotive Industry.csv
  - Source dataset used for analysis and dashboard creation.

- Exploring Trends in the Automotive Industry.twbx
  - Tableau Packaged Workbook that includes the workbook and related resources.

## How to Use

1. Open Tableau Desktop.
2. Open the file: Exploring Trends in the Automotive Industry.twbx.
3. Review dashboards, worksheets, and filters to explore trends.
4. If needed, inspect or update the source data in the CSV file.

## Notes

- Keep both files in the same project folder for easier management.
- If the workbook prompts for data source updates, reconnect it to the CSV file in this folder.

## Requirements

- Tableau Desktop (recommended for full editing experience).
